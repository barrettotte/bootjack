# bootjack

A minimal boot sector Blackjack.

## Running Locally

Build and run via QEMU with - `make qemu`

I also made a C implementation in [c/](c/) that can be run with - `gcc c/blackjack.c -Wall -o blackjack; ./blackjack`.
